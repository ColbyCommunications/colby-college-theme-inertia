# Full Bleed Image

The **Full Bleed Image** block displays a large image that spans the full width of the page with a content panel layered over it.

This block is designed to create a **strong visual section** that combines imagery with supporting text and optional calls to action.

---

## When to Use This Block

Use the Full Bleed Image block when:

- You want a visually striking section anchored by a large image
- You want to highlight a message alongside strong photography
- You want to include a call-to-action tied to a visual story

### Example uses:

- Promotional sections
- Featured programs or initiatives
- Campaign highlights
- Calls to action with supporting imagery

## Heading

The **Heading** field provides the main label or title for the content panel.

This text appears prominently within the overlay panel and is typically styled as an uppercase section heading.

Use short, descriptive headings that clearly communicate the purpose of the section.

---

## Paragraph

The **Paragraph** field contains the supporting description for the section.

This text should expand on the message introduced by the heading and provide additional context.

Keep this content concise for better readability over the image background.

---

## Buttons

Optional **buttons** can be added to create clear calls to action.

Buttons may link to:

- Related pages
- Programs or initiatives
- Event details
- External resources

Use buttons when you want visitors to take the next step after reading the content.

---

## Image

The **Image** field provides the large background image for the section.

The image fills the width of the page and serves as the primary visual element.

Choose images that:

- Are high resolution
- Have strong visual focus
- Provide enough contrast for the text panel to remain readable

---

## Layout Behavior

### Full Width Image

The image spans the full width of the page and can extend to the height of the viewport on larger screens.

---

### Content Overlay

The text panel appears over the lower corner of the image and contains the heading, paragraph, and optional buttons.

This panel ensures the content remains readable regardless of the image behind it.

---

## Tips for Best Results

- Use **high-quality images** that work well at large sizes
- Keep the **heading short and impactful**
- Limit paragraph length to maintain visual balance
- Use **one primary call-to-action** when possible
- Choose images that leave visual space for the content panel
