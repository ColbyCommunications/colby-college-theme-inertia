# Block Quote
The Block Quote is a typographic tool used to highlight key statements, testimonials, or significant excerpts from an article. It acts as a visual "break" in long-form content to draw the reader's eye.

## Field Configuration
In the WordPress editor, the Block Quote is intentionally streamlined to keep the focus on the content.

<Markdown>
{`
| Field | Type | Description |
| :--- | :--- | :--- |
| **Quote** | Textarea | The primary text of the quote. Supports multiple lines. |
`}
</Markdown>

### How to use it:
1. Add a new block and search for **"Block Quote"**.
2. Type or paste your text into the **Quote** field.
3. The block will automatically handle the quotation marks and styling based on our design system.


## Best Practices
To ensure the best reading experience, keep these guidelines in mind:

* **Length:** Aim for 1–3 sentences. Very long quotes can become difficult to read in this stylized format.
* **Placement:** Use a Block Quote every 4–5 paragraphs to maintain a good visual rhythm on the page.
* **No Manual Quotes:** Do not add your own quotation marks (`"`) inside the field. The design system adds these automatically to ensure consistent styling.


## Troubleshooting
**The quote looks too small/large:**
The Block Quote is designed to scale based on the amount of text. If you find the text is wrapping awkwardly, try breaking the quote into two paragraphs or slightly shortening the excerpt.
