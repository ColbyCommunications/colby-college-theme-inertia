# Full Bleed Hero

The **Full Bleed Hero** block creates a large, visually prominent section that spans the full width of the page.  
It is typically used at the top of a page to introduce important content with strong visual impact.

The block combines **headline content, optional buttons, and an image** in a centered layout.

---

## When to Use This Block

Use the Full Bleed Hero when:

- You need a strong visual introduction to a page
- You want to highlight key messaging
- You want to pair large editorial text with a supporting image

### Example uses:

- Page introductions
- Campaign or marketing headers
- Featured announcements
- Major program or event promotion

## Type

The **Type** setting controls the overall color theme of the hero.

### Dark

Uses a darker background pattern with light text.

Use this when:

- The image or page design benefits from stronger contrast
- You want a more dramatic or bold presentation

### Light

Uses a lighter background pattern with dark text.

Use this when:

- The page design calls for a lighter visual tone
- You want a softer or more minimal look

---

## Context

The **Context** section contains the primary text content of the hero.

This includes:

- Subheading
- Heading
- Paragraph text
- Optional buttons

Content is centered within the hero area to create a clear visual focal point.

---

## Image

The **Image** field adds a large supporting image below the hero content.

The image typically serves as a visual complement to the headline messaging.

Images may also include an optional **caption** that appears below the image.

Use images that:

- Reinforce the hero message
- Are high resolution
- Work well at large display sizes

---

## Layout Behavior

### Full Width Section

The hero spans the entire width of the page while keeping the content aligned to the site’s main layout grid.

---

### Centered Content

Text content is centered within the layout to create a strong focal point and improve readability.

---

### Responsive Image

If an image is included, it appears below the text and scales responsively across different screen sizes.

---

## Tips for Best Results

- Use **clear, concise headlines** for strong impact
- Pair the hero with **one primary call-to-action** when possible
- Choose images that work well at **large display sizes**
- Avoid overly long paragraphs in the hero section
- Use this block **sparingly**, typically once per page
